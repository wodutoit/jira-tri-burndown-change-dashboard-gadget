# TRI Sprint Dashboard Gadgets — project context

This is an Atlassian Forge app with 4 Jira dashboard gadgets (TRI Burndown, TRI Scope Change, TRI Rework, TRI Cycle Time), published on the Atlassian Marketplace. See `README.md` for developer setup and `USAGE.md` for what the gadgets actually do.

**Before changing anything that affects the Marketplace listing, manifest scopes, data handling, or third-party network calls**, read `.claude/MARKETPLACE-APPROVAL-GUIDELINES.md` first and check the change against it. Re-submission review can flag things that weren't a problem in dev.

For the architecture, the non-obvious design decisions, and the real bugs already found and fixed in this codebase (so they don't get reintroduced), read `.claude/PROJECT-CONTEXT.md`.
