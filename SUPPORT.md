# Support

Need help with any of the TRI Sprint Dashboard Gadgets (TRI Burndown, TRI Scope Change, TRI Rework, TRI Cycle Time)? Open an issue here:

**[github.com/wodutoit/jira-tri-burndown-change-dashboard-gadget/issues](https://github.com/wodutoit/jira-tri-burndown-change-dashboard-gadget/issues)**

## Before opening an issue

Check whether it's already been reported by searching [existing issues](https://github.com/wodutoit/jira-tri-burndown-change-dashboard-gadget/issues?q=is%3Aissue).

## What to include

To help us diagnose the problem faster, please include:

- Which gadget (TRI Burndown / TRI Scope Change / TRI Rework / TRI Cycle Time)
- What you expected to happen vs. what actually happened
- Your gadget's config, if relevant (Status → Phase Mapping, Sprint mode, SP field, etc.) — **do not** include your Jira site URL, sprint contents, or any other data specific to your organization
- A screenshot, if it's a display issue

## Feature requests

Feature requests are welcome — open an issue with a description of what you'd like to see and why.
